Hedge Accounting

Hedge relatonship

  • - can be effective or ineffective
    - consists of two sides: hedged and hedging

Hedging

  • - an strategy of minimizing potential investment losses
    - this is done by taking an offsetting position in a related derivative whose value changes inversely
    - the goal isn't increasing earnings but to protecting from different risks

Hedge effectiveness

  • - an extent to which a hedge relationship results in intended offsetting changes (fair value or cash flows)
    - effectiveness testing parameters: prospective, retrospective

Prospective

  • - value changes of the hedged item are expected to be offset almost 100% by the changes of the hedging instrument

Retrospective

  • - value changes of the hedged item must have been offset by 80%-125% by the change in the value of the hedging instrument

Hedges Supported

  • Foreign Exchange (FX) Cash Flow Hedge
    - used to hedge an anticipated future payment or receipt of a fixed amount of foreign currency
    - a cross currency swap is used to hedge a foreign currency debt or asset
    - testing for the hedge is based on the HDM*
    Interest Rate (IR) Cash Flow Hedge
    - allows hedging of interest rate risk of a floating rate debt or investment using a variable-to-fixed rate swap or interest rate options such as caps and floors
    - testing for the hedge can be based on fair value*
    Interest Rate Fair Value Hedge
    - used to hedge changes of fair value of a bond that is attributable to changes in a benchmark interest rate, such as LIBOR, i.e. an interest rate fair value hedge where fixed rate debt is swapped for floating rate debt
    - effectiveness is evaluated by comparing the change in value of the bond to the change in value of the swap
    Commodity Hedge
    - allows hedging of a commodity contract using a commodity forward
    - testing for the hedge can be based on fair value*
    *HDM = Hypothetical Derivative Method = testing for the hedge based on fair value
